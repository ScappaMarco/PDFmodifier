# PDFmodifier
PDFmodifier wants to be a website that allows to make change to **PDF files**; change as shrink, rotate and add/remove pages. The website backend logic is going to be made using **Apache PDFBox** (https://pdfbox.apache.org/).
The paging routing logic is made using **Spring-boot** (https://spring.io/projects/spring-boot), and the dependency management is made using **Apache Maven** (https://maven.apache.org/).
